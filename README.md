## 🛠 Краткое описание кейсов

### 1. Проверка гипотез в EdTech (A/B Тест)
Анализ эффективности системы наград "Daily Streaks" в рамках моей дипломной работы.
* **Результат:** Выявлен статистически значимый прирост времени сессии на **14.88%** ($p-value \approx 1.06 \times 10^{-11}$).
* **Методы:** Тест Шапиро-Уилка, T-критерий Уэлча, визуализация распределений.

### 2. SQL-аналитика магазина «Накарабине»
Построение полноценной аналитической инфраструктуры на базе спроектированных ранее бизнес-процессов.
* **Результат:** Реализованы автоматизированные скрипты для расчета воронки продаж и когортного анализа.
* **Техники:** Оконные функции (`FIRST_VALUE`), `DATE_TRUNC`, сложные Join-операции.

### 3. Технический аудит API (Otium)
Анализ логов производительности микросервисной архитектуры после внедрения новой функциональности.
* **Результат:** Обнаружены аномалии в задержках на эндпоинте поиска (p99 Latency) и рассчитан Error Rate.
* **Инструменты:** Pandas для агрегации логов, Seaborn для визуализации "длинных хвостов" распределения.
