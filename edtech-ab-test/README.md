## 📈 Проект: Анализ влияния геймификации в EdTech (A/B тест)

Данное исследование является практическим модулем дипломной работы на тему: **«Разработка ИТ-решения для повышения эффективности мобильных онлайн-обучающих приложений»**.

### 🎯 Бизнес-контекст

Необходимо было оценить, как внедрение системы ежедневных наград (Daily Streaks) влияет на вовлеченность. Цель — увеличить время нахождения пользователя в приложении.

### 🧪 Дизайн и методология

- **Группы:** Контрольная (A) и Тестовая (B) по 500 пользователей.
- **Метрика:** Average Session Length (ASL) — средняя длительность сессии в минутах.
- **Стек:** Python (Pandas, NumPy, SciPy, Matplotlib, Seaborn)
- **Статистический аппарат:** \* Проверка на нормальность (тест Шапиро-Уилка)
  - Сравнение средних значений (T-критерий Уэлча).

### 🚀 Результаты и выводы

- **Статистика:** Получено значение $p-value = 0.0004$. При уровне значимости $\alpha = 0.05$ нулевая гипотеза о равенстве средних отклонена.
- **Бизнес-эффект:** Среднее время сессии в тестовой группе выросло на **13.3%** (с 15.1 до 17.1 минут).
- **Рекомендация:** Внедрить функционал ежедневных наград для всех пользователей приложения.
