# Анализ эффективности геймификации в мобильном EdTech-приложении

Данный проект является практической частью моей дипломной работы на тему: **«Разработка ИТ-решения для повышения эффективности мобильных онлайн-обучающих приложений»**.

## 📌 Описание проекта
В исследовании проверяется гипотеза о влиянии геймификации (внедрение системы «Daily Streaks») на вовлеченность пользователей. Основной фокус сделан на применении математической статистики для принятия продуктовых решений.

## 🎯 Бизнес-гипотеза
* **Нулевая гипотеза ($H_0$):** система наград не влияет на среднюю продолжительность сессии пользователя.
* **Альтернативная гипотеза ($H_1$):** система наград статистически значимо увеличивает среднюю продолжительность сессии.
* **Целевая метрика:** Average Session Length (ASL) — среднее время в приложении за сессию (в минутах).

## 🛠 Технологический стек
* **Язык:** Python.
* **Библиотеки:** Pandas, SciPy, Matplotlib, Seaborn.
* **Инструменты:** Jupyter Notebook.

## 🧪 Методология исследования
1. **Подготовка данных:** формирование контрольной (A) и тестовой (B) групп по 500 пользователей.
2. **EDA (Исследовательский анализ):** визуализация распределений и проверка на выбросы.
3. **Статистическая проверка:**
   * Проверка распределения на нормальность (тест Шапиро-Уилка).
   * Применение **T-критерия Уэлча** для сравнения средних значений.
   * Расчет относительного прироста (Uplift).

## 📈 Основные результаты (на основе анализа)
* **Статистическая значимость:** полученный $p-value \approx 1.06 \times 10^{-11}$, что значительно ниже порога $\alpha = 0.05$. Нулевая гипотеза отвергнута.
* **Продуктовый эффект:** среднее время сессии в тестовой группе увеличилось с **15.04** до **17.28** минут. 
* **Относительный прирост (Uplift):** составил **14.88%**.
* **Рекомендация:** проект подтвердил высокую эффективность геймификации; рекомендовано внедрение функционала на всю аудиторию.
